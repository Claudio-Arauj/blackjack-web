# BlackJack Web
Projeto feito para trabalho da terceira unidade da matéria de Programação Web do curso de Sistemas de Informação - UFRN CERES

## Sobre o Projeto
Utilizamos a api [Deck of Cards API](https://deckofcardsapi.com) como base para a criação de jogos em um sistema web usando o framework NextJS para seu desenvolvimento.
